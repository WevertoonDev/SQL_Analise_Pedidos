# SQL Análise de Pedidos

Projeto de estudo em SQL utilizando banco de dados relacional para análise de pedidos e vendas.

## Objetivo

Praticar:

- Criação de tabelas
- Inserção de dados
- Relacionamentos entre tabelas
- Consultas SQL
- Análise de pedidos

## Estrutura do Projeto

- `Criando_tabelas.sql` → criação das tabelas
- `Inserindo_dados.sql` → inserção de dados
- `Pedidos.csv` → dados dos pedidos
- `Itens_de_Pedido.csv` → itens relacionados aos pedidos
- `consultas.sql` → consultas realizadas

## Tecnologias

- SQL
- Banco de Dados Relacional
- CSV

## Aprendizados

- Modelagem relacional
- Manipulação de dados
- Consultas analíticas
